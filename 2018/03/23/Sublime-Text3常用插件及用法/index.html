sublime text

Sublime-Text3常用插件及用法

[官方插件搜索网址]https://packagecontrol.io

Markdown Preview

作用：markdown预览插件
使用方法：
1.按住ctrl+shift+p调出命令面板，输入Preview,选择相应的预览选项；
2.按住ctrl+B进行构建生成html

emmet

作用：代码生成插件

sidebarEnhancements

作用：侧边栏增强

cTags

作用：代码跟踪

DocBlockr

作用：文档注释
使用方法：

ADBview

作用：查看console.log日志；用于android debug调试

html-css-js prettify

作用：代码美化
使用方法：
1.依赖node.js运行，在set node path中修改nodejs路径
2.set prettify preferences:
修改 “indent_inner_html”: true, “indent_scripts”: “normal”,
3.代码美化命令 ctrl+shift+H

BracketHighlighter

作用：该插件提供行数列高亮的各种配对的语法符号

Color​Picker

作用：调用window色板查看及设置css颜色
使用方法： ctrl+shift+c
